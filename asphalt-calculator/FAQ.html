Advanced Bituminous Calculator

What is the Advanced Bituminous Calculator?
The Advanced Bituminous Calculator is a comprehensive web-based tool designed to assist engineers, contractors, and planners in estimating costs and quantities for bituminous road construction projects. It aligns with the Department of Roads (DoR) Road and Bridge standards, ensuring calculations reflect industry norms. The calculator handles:
  • Material Costs: Based on volume, density, and rates.
  • Transportation Costs: Per m³ costs for tipper rent, driver, and fuel from two sources (Crusher 1 and Crusher 2).
  • Production Costs: Equipment usage costs (e.g., pavers, rollers).
It provides detailed tables and comparisons to optimize project planning and budgeting.
Why is the Advanced Bituminous Calculator necessary?
This calculator is crucial for:
  • Cost Accuracy: Prevents under- or over-estimation of project budgets with precise calculations.
  • Compliance: Adheres to DoR standards for bituminous road construction ensuring regulatory compliance.
  • Decision-Making: Compares costs from multiple material sources to choose the most economical option.
  • Resource Planning: Estimates material quantities and equipment needs for better project scheduling.
  • Time Efficiency: Automates complex calculations that would otherwise be time-consuming and error-prone.
Without such a tool, manual calculations risk project delays, cost overruns, and potential compliance issues.
What can I do with this calculator?
The calculator offers comprehensive functionality for road construction projects:
  1. Material Estimation: Calculate required quantities of materials (bitumen, aggregates, sand) for given compacted volumes.
  2. Cost Calculation:

How does the calculator work?
The calculator operates through three main functional sections:
  1. Material Quantity Estimation:
    • Input required compacted volume (m³) of material (BM, DBM, etc.)

  3. Production Cost Analysis:
    • Input equipment rates and usage hours
    • Calculate total production costs including materials
The system automatically updates results as inputs change, providing real-time feedback.

How do I calculate transportation costs per m³?
Follow these steps to calculate transportation costs:
  1. Click "Calculate Material Cost Per m³" to expand the section
  2. Enter values in the appropriate fields:
    • Material Cost per m³ at Source: Direct input from supplier quotes
    • Transportation Parameters:
      • Tipper specifications (capacity, rental rate)
      • Driver costs (daily wage)
      • Trip details (distance, expected trips per day)
      • Fuel parameters (price, vehicle mileage)
  3. Click "Calculate" to view detailed results

Calculation Formulas:

// Tipper Rent per m³ Tipper Rent per m³ = Tipper Rent per Day / (Total Tips × Tipper Capacity) // Driver Cost per m³ Driver Cost per m³ = Driver Cost per Day / (Total Tips × Tipper Capacity) // Fuel Cost per m³ Fuel Cost per m³ = (Round Trip Length × Diesel Price) / (Mileage × Tipper Capacity)

Example Calculation:

Input Values: - One Round Trip Length = 20 km - Diesel Price = 150 Rs/L - Mileage = 2 km/L - Tipper Capacity = 7.5 m³ Calculation: Fuel Cost per m³ = (20 × 150) / (2 × 7.5) = 200.00 Rs/m³

// Transportation cost calculation function function calculateTransportCost() { // Get input values const tripLength = parseFloat(document.getElementById('trip-length').value); const dieselPrice = parseFloat(document.getElementById('diesel-price').value); const mileage = parseFloat(document.getElementById('mileage').value); const capacity = parseFloat(document.getElementById('tipper-capacity').value); // Calculate fuel cost per m³ const fuelCostPerM3 = (tripLength * dieselPrice) / (mileage * capacity); // Update the display document.getElementById('fuel-cost-result').textContent = fuelCostPerM3.toFixed(2) + ' Rs/m³'; return fuelCostPerM3; }
