User Guide

  1. Go to your profile and click on Menu
  2. Go to Settings and Privacy
  3. Go to Account
  4. Download your data
  5. Click on All data
  6. Select Custom and Done
  7. Select Profile and posts and click Done
  8. Click on Request Data
  9. Wait for the download to be ready, then click on Download
  10. Click on Send code and receive the code in WhatsApp or email.
  11. Enter the code and click Continue
  12. Download will start automatically in the default browser. Click Open when finished.
  13. Extract the zip file.
  14. Open the Caputok App and click on Follower List
  15. Select Follower.txt from file manager.
  16. Click on Following List
  17. Select Following.txt from the file.
  18. Click on Analyze
  19. All non-followers will be displayed.
  20. Click on each user to check who unfollowed you.
Thank you.
